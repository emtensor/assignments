1. 작업 환경
 Windows 11 Education 64bit, AMD RYGEN 5 3600 6-Core, NVIDIA GeForce RTX 2060 SUPER, Visual Studio 2022 Debug x64

2. 실행 방법
 (1) src 폴더에 있는 파일을 프로젝트의 src에 복사한다.
 (2) data 폴더에 있는 파일을 프로젝트의 bin/data에 복사한다.
 (3) 프로젝트를 실행한다.

3. 기능
 - View->Set Left Hand on Wall: 좌선법
 - View->Set Right Hand on Wall: 우선법
 - q: 게임 모드 on/off
 - 게임 모드에서 방향키로 이동
 - i: 입구, 출구 설정
